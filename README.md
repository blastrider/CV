# Démarrage rapide GitHub Pages

Ce CV est un site statique (HTML/CSS/JS). Il est compatible GitHub Pages sans build.

## Option A — Site utilisateur `username.github.io`
1. Crée un dépôt public nommé `username.github.io`.
2. Ajoute `index.html` et `styles.css` à la racine du dépôt.
3. Va dans `Settings` → `Pages`.
4. Source: `Deploy from a branch`, puis branche `main` et dossier `/ (root)`.
5. Accède à `https://username.github.io` (publication en quelques minutes).

## Option B — Site de projet
1. Crée un dépôt (ex. `cv`).
2. Ajoute `index.html` et `styles.css` à la racine du dépôt.
3. Va dans `Settings` → `Pages`.
4. Source: `Deploy from a branch`, puis branche `main` et dossier `/ (root)`.
5. Accède à `https://username.github.io/cv`.

## Notes
- Si tu préfères, place les fichiers dans un dossier `docs/` et sélectionne `/docs` dans Pages.
- Les changements peuvent prendre jusqu’à 10 minutes pour être visibles.
